一、项目文件说明
============================================

项目文件夹在桌面，叫：
  📁 lzy的Javaweb作业 → 📁 library

里面重要文件：
  pom.xml          ← Maven项目配置文件（别动它）
  run.bat          ← 双击就能运行（不用IDEA也行）
  sql/init.sql     ← 数据库建表脚本（已经执行过了）
  src/             ← 所有Java代码和网页都在这

============================================
二、在 IDEA 中打开项目
============================================

第1步：打开 IntelliJ IDEA

第2步：点击顶部菜单栏 File → Open
       （或者点初始界面的 "Open" 按钮）

第3步：在弹出的窗口中找到这个路径：
       C:\Users\27125\Desktop\lzy的Javaweb作业\library

       选中 library 文件夹，点 "OK"

第4步：IDEA右下角会弹出一个蓝色小框：
       ┌──────────────────────────────┐
       │ Maven project found          │
       │ pom.xml has been detected    │
       │          [Load Maven Project]│
       └──────────────────────────────┘

       点击 "Load Maven Project" 按钮！！！
       （这个按钮很重要，不点的话项目跑不起来）

第5步：等待 IDEA 下载依赖（右下角有进度条），
       第一次可能需要 1~3 分钟（取决于网速），
       看到进度条消失就说明加载完成了。

第6步：确认项目结构是否正确：
       左侧项目树应该长这样：
       library
         ├── pom.xml
         ├── sql
         ├── src
         │   └── main
         │       ├── java/com/library
         │       │   ├── model
         │       │   ├── dao
         │       │   ├── util
         │       │   └── servlet
         │       └── webapp
         │           ├── WEB-INF
         │           ├── css
         │           └── jsp
         └── ...

第4步：点 OK 关闭

============================================
四、启动项目（任选一种方式）
============================================

【方式A】IDEA 内置运行（推荐，演示用）

第1步：IDEA 右侧边栏找到 "Maven" 标签
       （如果右侧没有，点 View → Tool Windows → Maven）

第2步：展开 library → Plugins → jetty
       找到 jetty:run，双击它

第3步：IDEA底部 "Run" 窗口会开始输出日志，
       等待看到这一行：

       [INFO] Started Jetty Server

       就说明启动成功了！

第4步：服务器会一直运行，直到你点红色方块停止按钮。


【方式B】双击 run.bat（省事，自动弹浏览器）

直接在文件夹里双击 library/run.bat 即可，
会自动启动服务器并在浏览器中打开登录页。


【方式C】IDEA Terminal 运行

第1步：IDEA 底部点 "Terminal" 标签

第2步：输入 mvn jetty:run 然后回车

第3步：等待看到 Started Jetty Server 即可

七、常见问题解决
============================================

Q1：双击 jetty:run 后，端口报错怎么办？
A1：因为之前没关服务器，端口被占了。
    去 IDEA 底部 "Run" 窗口，点红色方块停止旧的，
    然后重新双击 jetty:run。

Q2：IDEA 没有弹出 "Load Maven Project"？
A2：手动操作 —— 右键 pom.xml → Add as Maven Project

Q3：启动后中文显示乱码？
A3：IDEA 中 File → Settings → Editor → File Encodings
    把所有编码都改成 UTF-8。

Q4：浏览器打开 404 找不到页面？
A4：确认地址是 http://localhost:8080/library/login
    注意后面有 /library/login，不是只有 localhost:8080

Q5：数据库连不上，报 Communications link failure？
A5：检查 MySQL 是否在运行。
    按 Win+R，输入 services.msc，
    找到 MySQL80，确保状态是 "正在运行"。

Q6：mvn 命令找不到？
A6：确认 IDEA 的 Maven home path 设置正确（看第三章），
    或者直接在项目文件夹双击 run.bat。

八、答辩演示建议
============================================

1. 提前准备好：启动服务器、打开两个浏览器窗口
   （一个登录读者、一个登录管理员，方便切换）

2. 演示顺序建议：
   ① 读者登录 → 搜索 → 借书 → 查看借阅 → 还书
   ② 退出 → 管理员登录 → 自动进后台
   ③ 添加图书 → 编辑 → 查看借阅 → 管理员还书
   ④ 展示 @WebServlet 注解（代码层面）
   ⑤ 展示数据库表结构

3. 讲解重点：
   - 用了 Servlet + JSP 经典 JavaWeb 技术栈
   - 嵌入式 Jetty 服务器，不用配 Tomcat
   - DAO 设计模式，分层清晰
   - 借书还书的事务逻辑（库存 + 记录同步更新）

4. PPT 和代码要配合着讲：
   - 讲架构时切到 PPT 第5页
   - 讲数据库时切到 PPT 第6页
   - 讲功能时直接切到浏览器演示
